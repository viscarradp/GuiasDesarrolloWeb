Datos sobre la materia

Este curso introduce los fundamentos del desarrollo web, abarcando HTML, CSS y JavaScript. A través de ejercicios prácticos, los estudiantes aprenden a estructurar, diseñar y dar interactividad a sitios web modernos.

Contenidos del curso:

  • Unidad 1: Fundamentos de HTML
  • Unidad 2: Fundamentos de CSS
  • Unidad 3: Diseño con Flexbox y Media Queries
  • Unidad 4: Introducción a JavaScript
  • Unidad 5: Interactividad y DOM
